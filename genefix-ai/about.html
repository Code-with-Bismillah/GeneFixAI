About & Methodology

Methodology

  1. Data Collection and Preprocessing: Genome data is collected from public biological databases (e.g., NCBI, 2021) and preprocessed using bioinformatics tools to ensure high-quality, annotated sequences.
  2. Mutation Detection Using ML Models: Machine learning models (Libbrecht & Noble, 2015) detect SNPs, insertions, deletions, and other mutations by comparing subject DNA to reference genomes.
  3. Guide RNA Design: The system designs optimal gRNAs using a hybrid of traditional rules (PAM, GC content) and neural network predictions (J. Doench et al., 2014, 2016) to maximize precision and minimize off-targets.
  4. Simulation and Real-Time Monitoring: In-silico CRISPR edit simulations (J. Doench et al., 2014) are visualized for users. The system can integrate with lab sensors for real-time feedback to improve AI predictions.
  5. System Architecture and User Interface: GeneFix-AI is a modular web app with a Python/TensorFlow backend and a modern HTML/CSS/JS frontend. Users can upload genomes, receive mutation reports, and download gRNAs.
  6. Validation: The system is validated using benchmark CRISPR datasets and simulated experiments (Moreno et al., 2020), and refined with user feedback.
  7. AI-Powered Mutation Detection: Deep learning models (Angermueller et al., 2016) analyze DNA, classify mutation-prone regions, and predict genetic alterations.
  8. Guide RNA Design (AI): AI-based algorithms generate highly specific gRNAs, minimizing off-target effects (J. G. Doench et al., 2016).
  9. Real-Time Monitoring and Correction: The system tracks CRISPR-Cas9 edits in real time (Kleinstiver et al., 2016), using AI to suggest adjustments for optimal outcomes.
  10. User Interface and Interaction: Designed for accessibility, the UI allows researchers to upload data, run analyses, and receive actionable results (Norman, 2013).

BioData Integration

  • NCBI Entrez Integration: Direct search for genes, proteins, and sequences via /biodata/ncbi_entrez_search.
  • ENSEMBL API: Fetch gene annotations and comparative genomics data via /biodata/ensembl_gene.
  • UniProt Search: Retrieve protein-level information via /biodata/uniprot_search.
Example usage:
  • Search NCBI: ?term=BRCA1&db=gene
  • Ensembl gene: ?gene_id=ENSG00000139618
  • UniProt: ?query=TP53
